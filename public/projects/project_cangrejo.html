Cangrejo

Data mining tool to help run marketing campaigns.

Overview

Cangrejo is a data mining tool that will collect usefull marketing information about a website. Give it a list of websites and it will scrape email addresses, twitter statistics, site rankings, and other meta data for each website. You can then export this data into a spreadsheet to use in a marketing campaign.

Technologies
  • Node.js
  • Express.js
  • Postgres
  • Amazon EC2
  • Amazon RDS
  • Web Crawling
